Visualizations of Film Locations in San Francisco

Xiuping Tan   -   01/22/2018


Data Source: https://data.sfgov.org/Culture-and-Recreation/Film-Locations-in-San-Francisco/yitu-d5am

  The above dashboard uses data from DataSF about Film Locations in San Francisco from DataSF. It includes the titles, locations, fun facts, names of the director, writer, actors, and studio for filming locations of movies shot in S.F. starting from 1924. There are 1,622 rows in the dataset, which corresponds to one of the filming locations of S.F. in one movie. In the dataset, almost all the variables are categorical variables except the Released Year. My dashboard has three visualizations with a treemap, a line graph, and a bar graph.

  The top one is a treemap of top 15 filming locations in San Francisco. Speaking of locations, the first thing that comes to mind is the S.F. map. However, this dataset only provides general location but not standard address. We are interested in knowing some popular filming locations. The visualization used areas mark with for color and count as magnitude channels. The color goes from blue to teal as the number of movies increases. The more movies filmed in that locations, the larger in size. The main reason for choosing a treemap as visual technique is that location names can be long and are hard to display as labels. A treemap has large spatial areas for text.

  The line graph has a quantitative attribute using a line mark. Our interest in this visualization is to see the trends of how many movies filming in S.F. over time. The color of the line is blue, which is same as other visualizations. Since we only have categorical variables in our dataset, the only quantitative measure would be the number of data. Using one color is because no other measure values to compare with.

  The last one is the bar graph, which also displayed below using D3 visualization. It has two attributes using the horizontal spatial position channel for different directors (categories) and a line mark with vertical channel for the number of movies directed and filming in San Francisco. For color, I used one consistent color over all visualizations. The reason that I didn't use different colors for different directors is because it would look very distracting. I would like the audiences to focus on the values instead of coloring.

  Finally, I think my visualizations are successful at showing the data. The purpose of visualization is to provide insights and communicate findings. From my dashboard, it clearly tells which filming locations are most popular in San Francisco. It also finds the trend in number of movies filmed in S.F. over time. Suprisingly, there's a large rise in 2015. It also shows which director likes to film at S.F. the most. One thing can be improved is if we can transform the locations into address with latitude and longitude, we will be able to plot into a real San Francisco Map which easy to view.
